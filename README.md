# website-ticket-management
This is a part of job applicatiion.

## Usage

install database before use clone git command

clone git
```
    git clone https://github.com/sodtiruk/website-ticket-management.git
```
install dependency
```
    cd website-ticket-management
    cd client
    npm i
    cd ..
    cd server
    npm i
```
run server
```
    cd client
    npm run dev
```
run client
```
    cd server
    npm run dev
```
